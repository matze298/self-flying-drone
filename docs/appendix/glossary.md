# Glossary

This glossary expands recurring abbreviations used across the project docs. Keep entries short and link to deeper pages when a term becomes an implementation topic.

| Term | Meaning |
|---|---|
| ArduPilot | Open-source autopilot firmware project used here as the flight-safety authority. |
| ArduPlane | ArduPilot's fixed-wing vehicle firmware. |
| API | Application programming interface; a structured way for software components to call each other. |
| BEC | Battery eliminator circuit; a voltage regulator used to power electronics from the main battery. |
| CAN | Controller Area Network; a robust vehicle bus used for sensors and peripherals. |
| CG | Center of gravity; the aircraft balance point that strongly affects stability and control. |
| CI | Continuous integration; automated checks run by GitHub Actions or another build system. |
| CLI | Command-line interface; a terminal command intended for humans or scripts. |
| CRSF | Crossfire protocol; a serial RC/telemetry protocol also used by ELRS receivers. |
| DroneCAN | UAVCAN-derived CAN protocol used by many autopilot GNSS, compass, and sensor modules. |
| ELRS | ExpressLRS; open-source radio control link commonly used for RC transmitter/receiver setups. |
| EPO | Expanded polyolefin foam; common material for repairable RC aircraft airframes. |
| FC | Flight controller; the autopilot hardware running ArduPilot or similar firmware. |
| FPV | First-person view; live camera feed used for piloting or situational awareness. |
| GCS | Ground control station; QGroundControl, Mission Planner, MAVProxy, or similar operator software. |
| GNSS | Global navigation satellite system; GPS, Galileo, GLONASS, BeiDou, or multi-constellation receivers. |
| GPS | Global Positioning System; the US satellite navigation system, often used loosely for GNSS. |
| GStreamer | Multimedia pipeline framework used for camera capture, encoding, streaming, and recording. |
| H7 | STM32H7 microcontroller family commonly used in modern flight controllers. |
| JSON | JavaScript Object Notation; lightweight structured data format used for logs and examples. |
| MAVLink | Micro Air Vehicle Link; message protocol between autopilots, ground stations, and companion systems. |
| MAVProxy | MAVLink proxy/router and command-line ground station. |
| MAVSDK | High-level MAVLink software development kit used for vehicle clients and tests. |
| MP4 | MPEG-4 Part 14; common container format for recorded video files. |
| PDB | Power distribution board; board that distributes battery power to vehicle electronics. |
| PWM | Pulse-width modulation; common servo and actuator signal format. |
| RC | Radio control; manual pilot transmitter/receiver control path. |
| REPL | Read-eval-print loop; interactive shell for trying commands and inspecting results. |
| RF | Radio frequency; wireless link behavior including antennas, interference, and range. |
| ROS 2 | Robot Operating System 2; robotics middleware for multi-process robot systems. |
| RTL | Return to launch; failsafe or commanded mode that returns the aircraft toward its launch/home point. |
| SiK | Open-source telemetry radio firmware/hardware family commonly used for MAVLink links. |
| SITL | Software-in-the-loop; simulator mode running autopilot software on a normal computer. |
| TCP | Transmission Control Protocol; reliable network transport sometimes used for local MAVLink connections. |
| UDP | User Datagram Protocol; low-overhead network transport commonly used by MAVLink simulators. |
| UART | Universal asynchronous receiver/transmitter; serial interface used for GNSS, telemetry, receivers, and sensors. |
| UVC | USB Video Class; standard USB camera protocol supported by Linux without vendor-specific drivers. |
| V4L2 | Video4Linux2; Linux camera/video capture API used by UVC cameras and OpenCV/GStreamer. |
| VTOL | Vertical takeoff and landing; aircraft type that can take off and land vertically. |
| WFB-ng | Wi-Fi Broadcast next generation; digital video/telemetry transport used in some open FPV systems. |
